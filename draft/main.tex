\begin{abstract}
This is a template for an academic paper. It demonstrates how to structure sections, incorporate citations and equations, present tables and figures, and use cross-references throughout the document. All tables and figures are automatically generated using R and are updated each time the R code is re-run. So, there is no need to manually update the tables and figures. For additional information on how this works, please refer to the README file. This project works best when run using Cursor.
\end{abstract}
