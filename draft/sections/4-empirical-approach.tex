We estimate two linear regression models. The first one includes only vehicle weight:

\begin{equation}
    \label{eq:model_wt}
    mpg_{i} = \alpha + \beta_{1}\,wt_{i} + \varepsilon_{i}
\end{equation}

The second one includes both weight and horsepower:

\begin{equation}
    \label{eq:model_wt_hp}
    mpg_{i} = \alpha + \beta_{1}\,wt_{i} + \beta_{2}\,hp_{i} + \varepsilon_{i}
\end{equation}

The results of equations \ref{eq:model_wt} and \ref{eq:model_wt_hp} are shown in Table \ref{tab:main_results} in section \ref{sec:results}.
